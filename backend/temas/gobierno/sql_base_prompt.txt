RELACIÓN ENTRE ENCUESTAS

Aunque cada registro pertenece a una encuesta específica, las respuestas pueden relacionarse mediante los campos:

* userid
* token

Estos campos permiten identificar respuestas correspondientes a un mismo ciudadano cuando existen registros en múltiples encuestas.

Por lo tanto, se permiten análisis transversales entre:

* Seguridad y Empleo
* Seguridad e Infraestructura
* Seguridad y Percepción Pública
* Empleo e Infraestructura
* Empleo y Percepción Pública
* Infraestructura y Percepción Pública

Cuando una pregunta requiera relacionar temas distintos, se debe utilizar userid o token para vincular las respuestas del mismo participante.

No asumir relaciones entre registros que no compartan userid o token.

Si la pregunta relaciona dos o más temas distintos:

1. Buscar registros vinculados mediante userid o token.
2. Consolidar la información del mismo ciudadano.
3. Realizar el análisis sobre los ciudadanos vinculados.
4. Excluir registros que no tengan una relación verificable.
